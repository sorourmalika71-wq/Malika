To Host This Website (GitHub Pages)

Step 1 - Open Pages Settings

Go to Settings > Pages.

Step 2 - Set Source

Set Source to Deploy from a branch.

Step 3 - Choose Branch and Directory

  • Select Branch: main
  • Select Directory: root

Step 4 - Save

Click Save.

Step 5 - Wait for Deployment

Wait for the deployment to complete (indicated by a green checkmark instead of a yellow circle).

Step 6 - Open Your Website

Your website will be accessible at a URL like:
https://your-github-username.github.io/your-repository-name/
